# Веб приложение на TypeScript

## Техническое

приложение должно из себя представлять веб приложение-менеджер групп,оно будет работать в телеграме поэтому с дизайнами и тп лучше делай через API телеграмма.

для запросов API в бота и обратно используй энкодированный ip-address который в url запроса (http://myapp.com/index.html?replyip=http://myapp.com/api)

приложение должно включать в себя в самом низу панель управления,в которой будет следующее: Просмотреть свои задания,Просмотреть свои группы,Просмотр данных группы.
Для админов еще добавляется: Дать задание,Управление и создание групп,Просмотр всех групп

Вкладка просмотра своих заданий это просто таск-лист в котором на каждое из них можно кликнуть. 
при клике на задание переход на другую страницу где страница сначало запрашивает айди задания с апи бота и выводит данные такие как: номер задания,пользователь который выполняет задание,титул задания,проэкт задания,группа задания,информация о задании,время старта,время конца дедлайна.

Вкладка просмотра своих групп представляет из себя тот же таск лист но только с группами с аватарками на которые можно кликнуть и оно переведет пользователя на страницу с данными группы,и пользователями группы

Вкладка для админа выдачи задания представляет из себя форму которую можно будет отправить и имеет в себе опции выбора данных такие же как и просмотр задания

Вкладка для админа создания группы из себя представляет форму с именем группы,файл аватарки группы
